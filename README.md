# Bay Area GIS
GIS maps of the San Francisco Bay Area (Land Surface Temperature)

- NASA ECOSTRESS DATA
  - Land Surface Temperature (LST)
  - Accessible from [AppEEARS](https://appeears.earthdatacloud.nasa.gov/), or the ECO file in this repo
  
- Maps made with [QGIS](https://qgis.org/)
  - Following [I.C.E.C.R.E.A.M Tutorials](https://www.observingearthfromabove.com/tutorials.html), specifically [Tutorial 12](https://jeremydforsythe.github.io/icecream-tutorials/Tutorial12_MakingBetterMaps/Tutorial12_MakingBetterMaps.pdf)
